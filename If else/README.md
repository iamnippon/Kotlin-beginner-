# Number Guessing Game

Welcome to the Number Guessing Game! This is a simple console-based game built using Kotlin.

## How to Play

1. **Clone the Repository:**
   ```
   git clone https://github.com/iamnippon/Kotlin-beginner-.git
   ```

2. **Navigate to the Project Directory:**
   ```
   cd if-else
   ```

3. **Run the Game:**
   ```
   kotlinc Main.kt -include-runtime -d NumberGuessingGame.jar
   java -jar NumberGuessingGame.jar
   ```

4. **Instructions:**
   - You have to guess a randomly generated number between 1 and 100.
   - You will be provided feedback on whether your guess is too high, too low, or correct.
   - You have a maximum of 5 attempts to guess the number.

## Requirements

- Kotlin Compiler
- Java Runtime Environment (JRE)

## Development

This game is developed using Kotlin programming language. It demonstrates the following concepts:
- Input/output handling
- Random number generation
- Conditional statements (if-else)
- Recursion
